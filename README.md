# Predictive Modeling of Shooting Incident Fatality

This project analyzes historical NYPD shooting incident data and applies machine learning techniques to predict fatal outcomes.

## Tools & Technologies
Python
Pandas
NumPy
Scikit-learn
Power BI

## Dataset
NYPD Shooting Incident Historic Data

## Project Goals
Analyze patterns in shooting incidents
Build predictive models for fatality classification
Visualize insights using Power BI

## Files Included
NYPD_Shooting_Incident_Data_Historic.csv – dataset
NYPD_Shooting_Analysis.pbix – Power BI dashboard
Predictive Modeling of Shooting Incident Fatality Using Machine Learning.pdf – project report
